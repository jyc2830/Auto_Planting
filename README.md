# Auto Planting

Folia / Paper / Bukkit 계열에서 동작하는 자동 재식재 플러그인입니다.

## 동작

- 다 자란 주요 작물을 캤을 때, 왼손에 대응하는 씨앗/작물이 있으면 자동 재식재합니다.
- 재식재가 성공한 경우에만 왼손 아이템을 1개 차감합니다.
- 미성숙 작물은 일반 상태에서는 파괴되지 않고, `Shift`를 누른 경우에만 파괴됩니다.

## 지원 범위

- 밀
- 당근
- 감자
- 비트루트
- 네더워트

## 서버 호환성

- 단일 JAR로 배포합니다.
- 스케줄링만 서버 타입에 따라 분기합니다.
  - Folia: Region Scheduler 사용
  - Bukkit / Paper: 일반 Bukkit Scheduler 사용
- 이벤트 처리와 작물 판정 로직은 공통입니다.

## 빌드

```bash
mvn -DskipTests package
```

생성 파일:

- `target/auto-planting-1.0.0.jar`

## 설치

1. 빌드한 JAR를 서버의 `plugins` 폴더에 넣습니다.
2. 서버를 재시작합니다.
3. 왼손에 작물 씨앗이나 심을 수 있는 재료를 들고 수확하면 자동으로 다시 심어집니다.
