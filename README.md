# Auto Planting

Automatic crop replanting for Minecraft servers.

## Features

- Replants mature crops automatically when the matching item is in the offhand
- Prevents immature crops from breaking unless the player is sneaking
- Provides separate release jars for legacy, modern, and current server lines

## Downloads

- `AutoPlanting-2.0.0-legacy.jar`
- `AutoPlanting-2.0.0-modern.jar`
- `AutoPlanting-2.0.0-current.jar`

## Supported runtimes

- CraftBukkit
- Spigot
- Paper
- Folia

## Compatibility

| Jar | Minecraft versions | Runtime |
| --- | --- | --- |
| `legacy` | `1.12.x ~ 1.16.5` | CraftBukkit, Spigot, Paper |
| `modern` | `1.17.x ~ 1.21.x` | CraftBukkit, Spigot, Paper |
| `current` | `26.1.x` | CraftBukkit, Spigot, Paper, Folia |
